Entera: Загрузка документов

Обработка предназначена для заполнения документов в учетной системе 1с на основании распознанных сканов документов.

Список поддерживаемых обработкой конфигураций:

  • Бухгалтерия предприятия 3.0;
  • Управление торговлей 11.3 и 11.4;
  • Комплексная автоматизация 2;
  • различные отраслевые решения, выполненные на базе перечисленных конфигураций.

 

Общие сведения

Распознавание документов производится на портале Entera.pro. Пользователь имеет возможность загрузить отсканированный документ на портал как через данную обработку, так и в веб-интерфейсе портала.

Хранение документов на портале реализовано в разрезе Пространств, одна учетная запись может иметь несколько Пространств документов, что может быть использовано для загрузки документов, например, для различных организаций. Также обработка хранит установленные пользователем настройки в разрезе каждого Пространства портала.

 

Порядок работы

Для начала работы необходимо зарегистрироваться на портале, для входа в обработке используется существующая учетная запись.

Для получения списка существующих документов можно установить требуемый отбор:

  • По дате загрузки документа в сервис;
  • По дате документа
  • По ключевым словам

После нажатия на кнопку Обновить с портала будет загружен список документов, соответствующих отбору. При загрузке выполняется поиск уже существующих элементов справочников в 1с, в зависимости от установленных настроек (Номенклатура, Единицы измерения, Контрагенты, Организации).

Быстрый отбор заполненного списка можно выполнять по статусу документа используя фильтр в левой панели

Устанавливая флажки в левой колонке пользователь может выбрать распознанные порталом документы для заполнения документа в учетной системе. При нажатии на кнопку Создать документы будет выполнена проверка существования всех загруженных объектов и открыто окно для выбора пользователем либо существующих элементов справочников, либо для дальнейшего их создания в автоматическом режиме.

Создание элементов справочников:

В окне создания элементов доступен фильтр по всем/найденным/отсутствующим элементам, допускается удаление строк, содержащие элементы, создание которых не требуется. После нажатия на кнопку Создать объекты для строк с отсутствующим значением Соответствие в 1с будет выполнено автоматическое создание соответствующих элементов.

Отправка на распознавание:

При нажатии на кнопку меню Отправить документ пользователю будет предложено выбрать один или несколько файлов отсканированных документов для отправки. Количество распознаваемых в настоящий момент документов отображается в левой панели. Также допускается отправка файлов на распознавание их перетаскиванием на поле списка документов, например, из окна Проводника.

Настройки

Для удобства пользователя ряд значений, используемых для заполнения реквизитов по умолчанию при создании элементов вынесены в Настройки. Каждый элемент настройки имеет краткое и развернутое описание, отображаемое при выборе строки настройки.

Модуль кастомизации

Если есть особые требования к заполнению документов чтобы не вносить изменения в регулярно обновляемый модуль Энтеры, есть возможность использования модуля кастомизации: это внешняя обработка, хранящаяся в макете "МодульКастомизации" основной обработки. После заполнения в модуле объекта Модуля кастомизации процедуры-обработчика ОбработатьСобытиеОсновногоМодуля(ИмяСобытия,ПараметрыОбработки), ее следует сохранить и подключить в стандартный справочник внешних обработок. После этого в настройках основной обработки Энтера необходимо указать ее в настройке "Дополнительная внешняя обработка от Entera.pro для более гибкого заполнения документа" в разделе Системные настройки.
